Youtube Alarm

Thought of doing this due to my erratic schedule and my preference for waking up to some music. This website uses primarily Javascript and Youtube IFrame API to pause/play the video at the desired alarm time. Users have the added option of changing the wakeup music if they prefer something other than 'Shake It Off'. Feel free to make a pull request if you have any suggestions! Made for Desktop/PC Usage only at the moment

Special Project : Alumni-Network

Collaborators: Mallika Subramanian

An Alumni Network is often an essential part of any organisation and it was my pleasure to help set up one for a school in Kuwait. An interesting thing about this project was the budget constraints which eventually led to two version: static and server based. Each required drastically different organisational levels and proved to be an interesting challenge. We used Bootstrap 4 for the design and Flask for the web application. The entire process began using wireframe design modelling using Adobe XD, a tool I'd recommend to anyone who's starting out with web design. The Flask version has currently been put on pause but the Static version is up

Examination System

This was one of the reasons I decided on CS as a major. The actual application in itself wasn't too complex; it could read records, append new ones, sort them and so pretty much what's expected of a student academic record system. Built with login features and a neat Report Card printing option, this showed me how versatile coding really is. The simplest of tools we learnt in class were suddenly part of something much bigger; all thanks to a little creativity.



Game Library

Discovering the clrscr() function in C++ was the gateway to this project. I implemented 5 games in C++ using object oriented programming, trying my best to mimic GUI without the actual application creation process. The end result included scoreboards, menus, and other functionalities. Includes: Game of Fifteen, Maze, Tic Tac Toe, and more.
